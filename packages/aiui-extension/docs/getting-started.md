# Getting Started with @habemus-papadum/aiui-extension

The aiui intent tool as a Chrome extension: per-window side panel, channel binding, capture, ink, page tools.

## Install

```sh
npm install @habemus-papadum/aiui-extension
```

## Usage

```ts
import { greet } from "@habemus-papadum/aiui-extension";

greet("world"); // "Hello, world!"
```
